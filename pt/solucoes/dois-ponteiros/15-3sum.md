Precisamos encontrar todas as triplas que somam zero. A ideia e **ordenar o array** e usar a tecnica de dois ponteiros: para cada elemento, procuramos dois outros que completem a soma. Ordenar permite pular duplicatas e usar dois ponteiros eficientemente.

---

## âš™ï¸ Abordagem

1. Ordenar o array
2. Para cada elemento `nums[i]`:
   - Pular se for duplicata do anterior
   - Usar dois ponteiros: `esquerda = i + 1`, `direita = n - 1`
   - Enquanto `esquerda < direita`:
     - Calcular soma dos tres elementos
     - Se soma == 0: adicionar tripla, mover ambos ponteiros, pular duplicatas
     - Se soma < 0: mover `esquerda` para direita
     - Se soma > 0: mover `direita` para esquerda

---

- **Espaco**: O(1) ou O(n) dependendo do algoritmo de ordenacao
